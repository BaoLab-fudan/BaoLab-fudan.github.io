Tools
Open-source computational tools for multi-modal and spatial omics analysis

We develop and freely share software tools to help the broader research community analyze complex biological data. All tools are available on GitHub under open-source licenses.

Software
adMDM
An adaptive multi-modal data integration tool. adMDM learns joint representations from heterogeneous omics inputs, enabling robust integration across modalities while adapting to dataset-specific characteristics.
⌥ View on GitHub
SpatialMUSE
A tool for analyzing spatial transcriptomics and multi-omics data. SpatialMUSE integrates spatial context with gene expression profiles, supporting downstream tasks such as cell type deconvolution, domain identification, and cross-modal alignment.
